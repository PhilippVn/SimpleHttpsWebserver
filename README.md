#__Description__
Simple Webserver supporting HTTPS 1.1 running on port 80

#__Requirements__
- java jdk 17
- gradle

#__Build__
- Full Build: `./gradlew build`
- Only Jar: `./gradlew jar`

#__Run__
- `./gradlew run`
- `java -jar ./app/build/libs/<jarname>.jar`

#__Clean__
- `./gradlew clean`
